# Projet Développement Efficace

## Dépendances

### colorama
Librairie pour afficher du texte en couleur dans le terminal

## Structures de données utilisées
- Arbres
- Listes
- Piles
- Files

## Comment Jouer

Aller dans son terminal à la racine du projet et faire 
```python3 main.py```
Sinon, si votre IDE permet de lancer un fichier, aller dans ```main.py``` et le lancer

Bonne chance pour comprendre les menus :D


## Qu'est-ce que le projet

Notre projet est un jeu de stratégie dans lequel il faudra gérer nos ressources afin de pas crever, et de tabasser nos adversaires afin qu'ils crèvent
